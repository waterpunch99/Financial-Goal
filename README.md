# **FinGoal - 개인 재무 목표 관리 서비스**
---
## 프로젝트 개요

- 사용자가 자신의 재무 목표를 등록하고 목표 금액을 설정하고 입력해 기록하고 진행율을 확인 할 수 있습니다.

---

## 기술 스택
- Java / Spring Boot / JPA
- MySQL
---
## 주요 기능
- 목표(Goal) 등록, 수정, 조회
- 목표 진행률
---
## API 문서

- Swagger: http://localhost:8080/swagger-ui.html
